# Setup Infrastructure

This recipe sets up the core infrastructure for LearnFlow.

## Steps

1. Deploy Kafka on Kubernetes

Use skill: kafka-k8s-setup

2. Deploy PostgreSQL on Kubernetes

Use skill: postgres-k8s-setup

3. Deploy Dapr

Use skill: fastapi-dapr-agent (Dapr components only)

## Expected Result

Kafka, PostgreSQL, and Dapr are running on the Kubernetes cluster.
