# Modern UI System

## Purpose
Provide a consistent, premium UI foundation for all pages and components.

---

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion

---

## Design Principles
- Minimal, clean, SaaS-style UI
- Dark theme (default)
- Soft shadows and rounded corners
- Consistent spacing and layout
- Smooth animations

---

## Color System
- Background: #0B0F19
- Primary: #6366F1
- Accent: #22D3EE
- Highlight: #F59E0B
- Text: #E5E7EB

Use gradients and subtle glow effects.

---

## Typography
- Headings: bold (text-3xl → text-6xl)
- Body: clean and readable
- Proper spacing

---

## Core Components
- Navbar
- Footer
- Buttons (primary, secondary, gradient)
- Cards
- Form Inputs
- Badges
- Sidebar

---

## UI Rules
- Rounded corners (rounded-2xl)
- Hover + focus states
- Responsive design (mobile-first)
- Clean alignment and spacing

---

## Animations (Framer Motion)
- Fade-in on scroll
- Hover scale effects
- Page transitions
- Button animations

---

## Layout Guidelines
- Use container with max-width
- Maintain spacing consistency
- Avoid clutter

---

## Instructions
1. Apply design system globally
2. Replace old styles with new system
3. Ensure all components follow consistency
4. Keep code reusable and modular

---

## Validation Checklist
- [ ] Consistent design across app
- [ ] Responsive UI
- [ ] Smooth animations
- [ ] Clean layout and spacing

---

## Output Expectation
- Professional SaaS-level UI foundation
- Reusable components
- Visually consistent application
