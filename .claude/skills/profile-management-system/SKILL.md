# Profile Management System

## Purpose
Allow users to view and update their profile.

---

## Profile Page Features

### Show:
- Name
- Email
- Profile Picture
- User Level (based on XP/achievement)

---

## Edit Options
User can update:
- Name
- Email
- Password
- Profile Picture

---

## Password Rules
- Minimum 8 characters
- Must include letters and numbers

---

## Profile Picture
- Upload image
- Preview before saving

---

## Level System
- Display level (example: Level 3)
- Based on user progress/XP

---

## UI Requirements
- Clean card layout
- Edit button
- Save changes button
- Responsive design

---

## Instructions
1. Create profile page
2. Fetch user data from database
3. Allow editing fields
4. Save updated data
5. Show user level

---

## Output Expectation
- Functional profile page
- Editable user data
- Profile picture support
- Level displayed
