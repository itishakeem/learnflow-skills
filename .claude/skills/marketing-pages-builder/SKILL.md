# Marketing Pages Builder

## Dependency
Requires: modern-ui-system

---

## Pages to Create

### 1. Home Page
Include:
- Navbar with logo and links
- Hero section (headline + CTA)
- Features section (cards with icons)
- How it works section
- Call-to-action section
- Footer with social links

---

### 2. About Page
Include:
- Project introduction
- Mission and vision
- Platform explanation
- Clean structured layout

---

### 3. Contact Page
Include:
- Contact form:
  - Name
  - Email
  - Message
- Social media icons:
  - LinkedIn
  - GitHub
  - Twitter
- Optional contact info section

---

## Components to Use
- Navbar
- Footer
- Cards
- Buttons
- Form Inputs

---

## UI Requirements
- Clean spacing
- Section-based layout
- Consistent design system usage
- Responsive design

---

## Animations
- Section fade-in
- Hover effects on cards
- CTA button animations

---

## Folder Structure

/app
  /page.tsx
  /about/page.tsx
  /contact/page.tsx

---

## Instructions
1. Create all pages
2. Apply UI system styles
3. Use reusable components
4. Ensure responsiveness
5. Add animations
